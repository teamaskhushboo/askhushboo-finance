# #AS KHUSHBOO Finance 💛

**AI-Powered Brand Finance Management App**

> "Khushboo That Speaks for YOU"

## About

Personal finance management app for #AS KHUSHBOO, a Pakistani luxury fragrance brand. Track expenses, revenue, get AI-powered insights, and manage your brand's complete financial lifecycle.

## Features

- **Dashboard** - Key metrics, charts, and recent activity
- **Expenses** - Track all business expenses with auto-categorization
- **Revenue** - Track sales by perfume and source
- **AI Assistant** - Chat-based financial advisor (Roman Urdu + English)
- **Insights** - AI-powered spending analysis and savings suggestions

## Tech Stack

- Next.js 16 + TypeScript
- Tailwind CSS + shadcn/ui
- LocalStorage for data persistence
- z-ai-web-dev-sdk for AI features
- Recharts for data visualization

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
